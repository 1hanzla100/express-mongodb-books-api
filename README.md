# Express Restful API with JWT auth 


## Getting Started

### Built With
* NodeJS
* ExpressJS

### Routes

- #### book Routes
    - */api/book/:id* **GET**
    - */api/book* **GET**
    - */api/book* **POST**
    - */api/book* **PUT**
    - */api/book* **DELETE**

#### Install Dependencies 
```bash
$ npm init
```

```bash
$ npm install express jsonwebtoken mongoose nodemon morgan @hapi/joi body-parser cors
```

#### Start Server 
```bash
$ npm start
```


#### Environment Variable
```bash
DATABASE_URL=
PORT=
```
